# Fake News Detection using Machine Learning

## Overview

This project detects whether a news article is Fake or Real using Natural Language Processing (NLP) and Machine Learning.

## Features

* Data preprocessing
* TF-IDF Vectorization
* Passive Aggressive Classifier
* 99.44% Accuracy
* Predicts Fake or Real news

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Dataset

* Fake.csv
* True.csv

## How to Run

1. Open the notebook in Google Colab.
2. Upload Fake.csv and True.csv.
3. Run all cells.
4. Enter a news article to test the model.

## Result

Model Accuracy: **99.44%**
